# College Scorecard: Higher Education Outcomes

## Project Overview
This project analyzes whether paying more or attending a more selective school 
leads to better student outcomes using the U.S. Department of Education 
College Scorecard dataset.

## Research Question
Do expensive, selective schools consistently produce better financial results, 
or do some institution types offer a stronger return on investment?

## Key Metrics
- Inputs: Tuition cost, SAT scores / admission rate (selectivity)
- Outputs: Graduation rate, median earnings 10 years after enrollment

## Setup
1. Clone this repo
2. Run `pip install -r requirements.txt`
3. Download the dataset from https://collegescorecard.ed.gov/data/
4. Place the CSV in the `data/` folder
5. Open `notebooks/01_cleaning_eda.ipynb` in Jupyter

## Tools Used
- Python, Pandas, Matplotlib, Seaborn, Plotly, Streamlit
